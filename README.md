# Audited TodoList
Spring Boot todo list with Auditing

Usage (with eclipse):
1.) Clone the project
2.) Eclipse: File -> Import -> Maven -> Existing Maven Projects
3.) Run
4.) Navigate to localhost:8080

Käyttätunnukset ja Salasana<br>
Käyttätunus:user<br>
Salasana: user<br>
Rooli: Peruskäyttäjä

Käyttätunus:admin<br>
Salasana: admin<br>
Rooli: admin
